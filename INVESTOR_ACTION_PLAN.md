# Plan de Acción para Subir Invertibilidad

## Objetivo

Mover Civitas Capital desde "proyecto técnico ambicioso" a "oportunidad de inversión con foco comercial".

## Acciones inmediatas

### 1. Foco

Reducir el mensaje a Civitas Pro:

> Cualificación financiera de compradores inmobiliarios para agencias, brokers y promotoras.

### 2. Demo

Crear una demo que muestre:

- Lead comprador.
- Capacidad real.
- Inmuebles compatibles.
- Simulación hipotecaria.
- Servicios recomendados.
- Dinero potencial.

### 3. Prueba comercial

Preparar contacto con:

- Agencias medianas.
- Brokers hipotecarios.
- Promotoras pequeñas/medianas.

### 4. Métricas

Medir:

- Leads cualificados.
- Tiempo ahorrado.
- Conversión a visita.
- Conversión a solicitud hipotecaria.
- Servicios contratados.

### 5. Repo

Antes de subir:

- Mantener privado.
- Ignorar `target/`.
- Revisar secretos.
- Añadir docs de arranque.
- Evitar promesas de garantía financiera.

## Qué debe ver un inversor

- Problema concreto.
- Comprador claro.
- Demo simple.
- Modelo de ingresos.
- Roadmap realista.
- Capacidad técnica.
- Señales de validación.

## Qué no debe ver

- Mensaje disperso.
- Promesas imposibles.
- Demasiada arquitectura sin demo.
- Repositorio con binarios generados.
- Falta de foco comercial.
