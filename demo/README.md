# Demo Civitas Pro

Esta carpeta contiene datos sintéticos para construir una demo de inversor o piloto comercial.

## Historia demo

Una agencia recibe compradores interesados, pero no sabe cuáles son financiables. Civitas Pro calcula capacidad de compra, clasifica la viabilidad, cruza inmuebles compatibles y recomienda servicios.

## Archivos

- `civitas-pro-demo-data.json`: dataset sintético con compradores, zonas, inmuebles, productos financieros y servicios.
- `civitas-pro-demo.html`: demo visual ejecutable en navegador sin servidor.

## Uso recomendado

1. Cargar los compradores.
2. Calcular cuota máxima prudente.
3. Estimar precio máximo de compra.
4. Comparar contra inmuebles.
5. Mostrar financiación orientativa.
6. Recomendar servicios de cierre.

Los datos son ficticios y no deben usarse como recomendación financiera real.

## Abrir demo

Abrir `demo/civitas-pro-demo.html` en el navegador.
